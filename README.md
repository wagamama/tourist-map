# tourist-map
Plot positions of toilet, trash can, free wifi and drinking water on map for tourists visiting Taipei

# Data source
[臺北市公共場所飲水機資訊](http://data.taipei/opendata/datalist/datasetMeta?oid=f158e1bc-8400-44c2-a93a-e09361ddb7f5)

[臺北市行人專用清潔箱](http://data.taipei/opendata/datalist/datasetMeta?oid=a835f3ba-7f50-4b0d-91a6-9df128632d1c)

[臺北市公眾區免費無線上網熱點資料(新版)](http://data.taipei/opendata/datalist/datasetMeta?oid=6aa6532d-652f-4c1b-814a-4646b75407af)

[臺北市公廁點位資訊](http://data.taipei/opendata/datalist/datasetMeta?oid=8a4bc326-72f8-4ddf-8d57-9881a528992e)
